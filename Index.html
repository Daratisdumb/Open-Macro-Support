XTernal Troubleshooting

Installation & Update Errors

The Issue: The macro asks you to update, downloads the files, but always launches the old version again.

The Fix: The GitHub link for the auto-updater is currently suspended, causing the download to silently fail. You must install the new version manually:

  1. Delete your current XTernal folder completely. (Your settings are saved safely elsewhere!)
  2. Download the newest .zip file provided in the Discord releases channel.
  3. Right-click the downloaded .zip file and select Extract All...
  4. Open the new extracted folder and run the macro directly from there.
[Image: Right-click > Extract All context menu]

The Issue: You try to run the macro, but Windows says the script file is missing or cannot be found.

The Fix:

  1. Extract properly: Ensure you extracted the .zip file. Running the macro while it is still trapped inside the compressed zip folder will cause it to crash.
  2. Check Windows Defender: Antivirus software often mistakes memory-reading macros for viruses. Open Windows Security > Protection History, find the recently blocked threat, and select Restore or Allow on device.

The Issue: You get a weird popup showing "PK" and a bunch of strange rectangle symbols.

The Fix: This happens if you accidentally download the .zip update file and try to rename it to an .ahk file to force it to run.

  1. Delete the broken file you renamed.
  2. Download a fresh .zip from the release channel.
  3. Right-click the .zip, select Extract All..., and run the real Main.ahk script inside the new folder.
[Image: The Line 1 error box]

The Issue: The macro's settings file got wiped blank by a PC crash, power loss, or forced closure.

The Fix: Delete the broken file so the macro can build a fresh one.

  1. Press Windows Key + R to open the Run box.
  2. Type %appdata%\OpenMacro\XTernal and hit Enter.
  3. Delete the settings.json file.
  4. Restart the macro. It will automatically create a healthy settings file!

Connection & Roblox Errors

The Issue: The macro cannot read your game's memory to see that you have spawned in.

The Fix:

  • Run as Administrator: Close the macro, right-click the file, and select Run as Administrator to bypass Windows security blocks.
  • Use the Web Version: The macro will not work on the Microsoft Store or Xbox App versions of Roblox. Launch the game directly from your web browser (roblox.com).
  • Force Reconnect: Click into your game so your character can move, and press F3 to force the macro to reconnect.

The Issue: Error 5 translates directly to "Access Denied." Windows is actively blocking the macro from looking inside Roblox.

The Fix: Close the macro entirely. Right-click the macro file and select Run as Administrator.

The Issue: Error 31 means your computer is still launching a hidden Microsoft Store or Xbox version of Roblox, which the macro cannot read.

The Fix:

  1. Open Roblox normally and press Ctrl + Shift + Esc to open Task Manager.
  2. Find Roblox, right-click it, and select Open file location.
  3. If the folder path says WindowsApps, delete the shortcut you used to launch it. Go directly to roblox.com and install the browser version.

Minigame & Fishing Glitches

The Issue: The minigame starts, but the white bar flings way past the fish and instantly fails.

The Fix: The macro calculates fish movement based on your frame rate. If your game stutters or lags, the macro over-corrects. Optimize your PC:

  1. In Roblox, set your Graphics Quality to 1 and zoom your camera all the way in to boost FPS.
  2. Turn off the macro's live stat overlay (the transparent black window) to save processing power.
  3. Open the macro Settings > Adjustments, lower your Prediction Strength (try 3.0), lower Proportional Gain (try 0.30), and increase Velocity Damping (try 45).
[Image: The Adjustments/Settings menu with the recommended numbers]

The Issue: The macro hooks the fish, but refuses to pull the bar to the right.

The Fix: Your mouse inputs are being hijacked or your screen math is broken.

  1. Unplug Controllers: Unplug any Xbox/PlayStation controllers or racing wheels. Roblox prioritizes them over the macro's simulated clicks.
  2. Fix Display Scaling: Right-click your Windows Desktop and select Display Settings. Make sure Scale and layout is exactly 100%. Anything higher breaks the screen math.

The Issue: The macro fights the fish perfectly, but suddenly gives up and lets the bar fall.

The Fix:

  • CPU Overload: If your laptop hits 100% CPU usage during a chaotic moment, Windows briefly freezes the macro, dropping the mouse click. Close all background apps like Discord or Spotify.
  • Standard Rod: Make sure you are not using an enchanted rod (like Swift or Resilience). Enchantments change the physics of the minigame and can break the macro's math. Try a standard Flimsy or Plastic rod first.
